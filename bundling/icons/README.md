# bundling/icons/

Application icons referenced by `appinfo.ini`.

Motif: gear + sparkle on a rounded-square #0A0A0A container,
accent #3B82F6 / #60A5FA. Generated by `bundling/generate_icons.py`
from a 512px Pillow-rendered master, then downscaled with LANCZOS.

Files (PortableApps.com Format):

- `appicon.ico`     — multi-resolution ICO (16/32/48/64/128/256 px)
- `appicon_16.png`
- `appicon_32.png`
- `appicon_75.png`  — used by the PortableApps menu
- `appicon_128.png`

To regenerate: `python bundling/generate_icons.py`
